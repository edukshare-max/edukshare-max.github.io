🎯 Resumen Ejecutivo

El presente informe describe la implementación exitosa del Sistema de Atención en Salud Universitaria de la Universidad Autónoma de Guerrero, un proyecto de innovación tecnológica desarrollado por la Dirección de Innovación en Salud Universitaria del Centro de Investigación Transdisciplinar.

Este sistema representa un avance significativo en la digitalización de servicios universitarios, permitiendo a estudiantes, académicos y personal administrativo acceder de manera segura y eficiente a sus credenciales institucionales y expedientes de salud a través de una plataforma web moderna y accesible desde cualquier dispositivo.

📋 Contexto del Proyecto

La Universidad Autónoma de Guerrero, en su compromiso con la modernización y la excelencia académica, ha desarrollado este sistema como parte de su estrategia de transformación digital. El proyecto surge de la necesidad de optimizar el acceso a servicios de salud universitaria y facilitar la gestión de información académica.

Bajo la dirección de la Dirección de Innovación en Salud Universitaria, este sistema integra tecnologías de vanguardia para proporcionar una experiencia de usuario excepcional, garantizando la seguridad, privacidad y accesibilidad de la información.

✨ Características Destacadas

  • Interfaz Moderna e Intuitiva: Diseño atractivo con los colores institucionales de la UAGro, optimizado para una experiencia visual excepcional.
  • Accesibilidad Multiplataforma: Compatible con dispositivos móviles, tablets y computadoras de escritorio.
  • Seguridad Avanzada: Cifrado de contraseñas con bcrypt, tokens JWT y protección CORS.
  • Rendimiento Optimizado: Sistema de caché y retry logic para garantizar disponibilidad incluso con conexiones intermitentes.
  • Escalabilidad: Arquitectura en la nube con Azure Cosmos DB y servicios de backend en Render.
  • Promociones de Salud: Sistema integrado para difundir información relevante sobre salud y bienestar universitario.

💻 Stack Tecnológico

Frontend

Flutter Web: Framework de Google para desarrollo web multiplataforma con rendimiento nativo y diseño moderno.

Provider: Gestión de estado eficiente y reactiva.

Despliegue: GitHub Pages con dominio personalizado app.carnetdigital.space

Backend

Node.js + Express: Servidor de alto rendimiento con API RESTful.

JWT: Autenticación segura basada en tokens.

bcryptjs: Cifrado de contraseñas con salting.

Despliegue: Render Cloud Platform

Base de Datos

Azure Cosmos DB: Base de datos NoSQL distribuida globalmente de Microsoft Azure.

Contenedores: carnets_id, cita_id, promociones_salud, usuarios_matricula

Rendimiento: 400 RU/s con escalabilidad automática

Seguridad

CORS: Control de acceso entre orígenes configurado.

Helmet: Protección de headers HTTP.

Rate Limiting: Prevención de ataques DDoS.

HTTPS: Comunicación cifrada end-to-end

🔒 Seguridad y Privacidad

El sistema implementa múltiples capas de seguridad para proteger la información sensible de los usuarios: cifrado de contraseñas con bcrypt (10 rounds), tokens JWT con expiración automática, validación de identidad contra la base de datos institucional, y cumplimiento con estándares de protección de datos personales.

🌱 Sostenibilidad y Medio Ambiente

El proyecto contribuye significativamente a la sostenibilidad institucional al eliminar la necesidad de imprimir carnets físicos. Se estima que el sistema evitará la impresión de más de 10,000 carnets de PVC anualmente, reduciendo el impacto ambiental y los costos operativos. Además, la digitalización de expedientes médicos reduce el uso de papel en las clínicas universitarias.

Implementación y Desarrollo Futuro

⏱️ Línea de Tiempo del Proyecto

Fase 1: Planificación (Completada)
Análisis de requerimientos, diseño de arquitectura y definición de alcance del proyecto.
Fase 2: Desarrollo (Completada)
Implementación del frontend en Flutter, backend en Node.js y base de datos en Azure Cosmos DB.
Fase 3: Pruebas (Completada)
Testing exhaustivo en localhost, validación de seguridad y pruebas de usuario.
Fase 4: Despliegue (Completada)
Publicación en producción con dominio app.carnetdigital.space, configuración de SSL y monitoreo.
Fase 5: Piloto (En Curso)
Proyecto piloto con grupo selecto de usuarios para validación y feedback.

🔐 Aspectos de Seguridad

Protección de Datos

El sistema cumple con las normativas de protección de datos personales, implementando cifrado end-to-end y almacenamiento seguro en Azure Cloud.

Autenticación

Sistema de autenticación robusto con JWT, expiración automática de sesiones y validación de identidad contra base de datos institucional.

Auditoría

Registro completo de accesos y modificaciones, permitiendo trazabilidad de todas las operaciones realizadas en el sistema.

Respaldos

Respaldos automáticos diarios en Azure Cosmos DB con retención de 30 días y recuperación ante desastres.
